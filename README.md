# MU42022_Code
Code for manuscript, "Roseobacter enrichment early in life facilitates future colonization of Roseobacter bacteria and improves long-term survival against Vibrio aestuarianus subsp. francensis in the Pacific oyster", Wright-LaGreca et al., 2025 (in prep as of Dec, 2025). 

Portions of code was generated using open AI software (Chat GPT, Claude). AI-generated code was edited by the authors and code function and use was validated by the corresponding author (MWL). Statistical decisions were validated using outside sources and cited in the manuscript.
